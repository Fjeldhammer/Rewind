The Code

                    
                        function getValues() {

                            let userString = document.getElementById('userString').value;
                        
                            let revString = reverseString(userString);
                        
                            displayString(revString);
                        }
                        
                        function reverseString(input) {
                        
                            let revString = '';
                        
                            for (let index = input.length - 1; index >= 0; index = index - 1) {
                        
                                let letter = input[index];
                        
                                revString = revString + letter;
                            }
                        
                            return revString;
                        }
                        
                        function displayString(output) {
                        
                            document.getElementById('results').textContent = output;
                        
                            let alertBox = document.getElementById('alert');
                        
                            alertBox.classList.remove('invisible');
                        }

The Code is structured in three functions.

getValues

The function getValues retrieves the data entered into the program at the user interface, appropriates it for use in the next function and passes it forward.

It assigns this data value the name 'userString'. Then declares a variable 'revString' which will pass the userString data to the next function. And then sets up the pass with 'reverseString(userString)' for moving the data value to the next function.

This function will later receive the output from the second function and prep this new to be passed to the third function.

reverseString

The function reverseString calls the data value entered in the first function (input), and declares the variable 'revString'. Using a for loop, it iterates each character from the userString (array) through the function in order to reverse the order of the characters in the userString and passes it to the next function to display the results.

displayString

The function displayString retrieves the output from the reverseString function (revString results) and packages it in the document.getElementById().textContent alert.

Then it is passed to the html page (view) to display to the user in the previously hidden alertbox.
